### About
A Laravel dashboard demo.

### Usage
1. clone this project
2. install composer
3. run:
```bash
composer install
```
4. config database, file path: <code>config/database.php</code>
5. config Laravel enviroment, run:
```bash
cp .env.example .env
php artisan key:generate
```
then, use this key in .env
